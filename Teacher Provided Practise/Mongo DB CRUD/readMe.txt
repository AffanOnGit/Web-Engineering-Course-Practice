Download Mongodb Compass from here: https://www.mongodb.com/try/download/compass
Download the project folder given below, install the following npm packages/dependencies for the project: 
    "body-parser",
    "dotenv",
    "express",
    "jsonwebtoken",
    "mongodb",
    "mongoose"
Run the project using node app.js or nodemon app.js
To test the API endpoints, I have added the curl commands at the end of app.js file.
